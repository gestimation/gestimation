# Story and Quiz − Study design IV

【キーワード: OS/PFS/DFS/奏効率, 交絡/情報バイアス/選択バイアス, 観察研究】　【難易度: ★】

### 調査で気をつけておくべきバイアス

**私「お父さん、ハード なエンドポイントとソフト なエンドポイントって言い回しがあるじゃない？」**

お父さん「あるね」

**私「臨床試験にあこがれてる先輩がいてね、復職状況はソフトなエンドポイントだからだめだ、OSはハードなエンドポイントだからいいんだ、みたいな言い方をするのよ。否定されてるみたいなんだけど、どういうことなのか、よくわかんなくて」**

お父さん「確かにがん臨床試験はOSをエンドポイントにすることが多いけど、それを基準にダメ出しするのもどうかと思うけどね。それに、慣用的にハード・ソフトっていう表現が用いられるけど、その意味を正確に伝えるなら、客観的・主観的って言った方がいいんじゃないかな」

**私「客観的？主観的？」**

お父さん「よくある議論を挙げると、OSはハード、PFS（無増悪生存期間）はソフトというのがある。OSは死亡日と最終生存確認日だけで決まるから、ハードなエンドポイント。PFSでは、死亡とがんの増悪がイベントとして定義されるんだけど、そうするとデータを集めるために増悪日の情報が必要になる。ところが臨床的には増悪がいつ起きたかは、主観的に判断されるケースがあるよね。全身状態や腫瘍マーカーが悪化したりするケースだね。臨床試験で、こういったケースをイベントとみなすかは議論がわかれるけど、こういった意味でPFSはソフトなエンドポイントって言われることがある」

**私「なるほど、ハード=客観的、ソフト=主観的ってことね。たしかに、私の調査票はソフトなエンドポイントって言われるわけだ。そもそも患者さんの主観を尋ねたいんだもんね」**

お父さん「まあ、その先輩が伝えたかったのは、復職状況を調べるうえで、できるだけあいまいな部分を排除して、客観的に測定しなさいってことじゃない？調査対象者が、退院後の就労日を正確に回答してくれるなら、十分に客観的な測定ができていると思うけどね」

**私「じゃあさ、ハードなエンドポイントにはバイアスがない、ソフトなエンドポイントにはバイアスがあるって考えていい？」**

お父さん「主観が入るとバイアスが生じやすいとは言えるけど、そんなに単純じゃないかな。調査がいい加減だったら、死亡が完全に特定できず、死亡率を過小評価することだってあるもの」

**私「じゃあ、私の調査で、それ以外に気をつけておいた方がいいバイアスってある？」**

お父さん「調査票を郵送するんでしょ。そういうとき、回答してくれない患者が多いと偏りが生じやすいから、回答率を挙げる工夫をするといい。たとえば、調査票で年収って聞いてる？」

**私「聞くかもしれない」**

お父さん「経済状況のようなナイーブな質問内容には十分配慮した方がいいよ。それが理由で回答してくれないと、解析対象集団が調査対象全体を代表していると言えなくなってしまう。他にはね、ストーマ非保有者よりストーマ保有者の方が、調査票への回答率が高くなっちゃうとか。そもそもストーマ非保有者とストーマ保有者の回答率に差があると、これらの集団を比較してもバイアスがあるって批判されるかもしれない」

**私「それって交絡のこと？」**

お父さん「交絡もバイアスの一種だけど、違う。これは母集団からのサンプル抽出に影響する”選択バイアス”といって区別されるんだ。それに、年収って尋ねられると、過大申告してしまいがちだよね。このように集めた情報自体に偏りがあることを、”情報バイアス”と呼んでいる。調査方法について授業するときは、選択バイアス、情報バイアス、交絡の3つに気をつけなさいって教えるようにしている。一部の対象者ばかり選ばれないようにしなさい、情報を集めるときはバイアスが入らないように配慮しなさい、集団を比較するときは、比較したい要因以外の特徴を揃えるようにしなさいってね」

**私「ふーん。やっぱり臨床研究って言葉遣いがよくわかんないな。まあいっか」**

::: callout-note
<img src="https://www.notion.so/icons/alert_blue.svg" alt="https://www.notion.so/icons/alert_blue.svg" width="40px"/> **ランダム誤差とバイアス**

統計学は誤差を扱う学問ですが、臨床研究ではランダム誤差（random error）とバイアス（bias）を区別することが大切です。ランダム誤差は、基本的にはゼロを中心としたばらつきを想定しています。一方で、得られた推定値が真の値から系統的にずれていることを、バイアスといいます。臨床研究における統計学の目標は、ランダム誤差を制御し、バイアスを可能な限りゼロに近づけることです。

![図1.png](Story%20and%20Quiz%20%E2%88%92%20Study%20design%20IV/%25E5%259B%25B31.png)
:::

::: callout-note
<img src="https://www.notion.so/icons/help-alternate_blue.svg" alt="https://www.notion.so/icons/help-alternate_blue.svg" width="40px"/> **最後にクイズです**

ランダム誤差とバイアスへ対処するために様々な工夫がなされますが、それを検討するタイミングは、研究計画段階と統計解析段階にわけることができます。これを踏まえて、臨床研究の考え方として正しいのは、次のうちどれでしょうか。

1.  ランダム誤差は統計解析を工夫すれば対処できる。だから研究デザインではバイアスについて検討すべき
2.  ランダム誤差もバイアスも統計解析を工夫すれば対処できる
3.  統計解析段階では工夫するにも限界がある
4.  臨床試験では研究デザインが重要。観察研究では統計解析が重要
:::

::: {.callout-note collapse="true" title="解答を表示"}
-   **正解は3です。**

サンプルサイズが小さすぎることが、ランダム誤差の一番の原因です。こう考えると、ランダム誤差・バイアスのどちらも、研究計画段階でじゅうぶんに検討しておかないと、後から取り返しがつかないかもしれません。
:::
